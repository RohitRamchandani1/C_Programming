# Installing an Integrated Development Environment(IDE)
- IDE is a software that contains all the essential tools to develop, compile, link and debug code.
- In some cases, the user may have to integrate compiler and debugger tools manually.
- VSCode or STM32CubeIDE(Eclipse based IDE) can be used to write, compile, debug applications on STM32 ARM based microcontrollers.
- STM32CubeIDE is an eclipse IDE with STM32 related customization.
- STM32CubeIDE can be downloaded or the VSCode for STM32Cube can be downloaded for the development.
